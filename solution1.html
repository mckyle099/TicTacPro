If we're called first, there will be 0 (even number) squares played, and we are X
If we're called second, there will be 1 (odd number) square played, and we are O
If we're called third, there will be 2 (even number) squares played, and we are X
...etc
As you can see, for any given turn, if there's an even number of squares played, we're X, otherwise we're O

function getSymbol(board)
  count = 0
  for i=1,9 do
    if board[i] != '\0' then
      count = count + 1
    end
  end
  if count % 2 == 0 then
    return 'X'
  else
    return 'O'
  end
end

We use the modulo operator (%) to divide by 2 and check for remainder.
